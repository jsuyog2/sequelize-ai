# Contributing to sequelize-ai

First off, thanks for taking the time to contribute! 🎉

## Getting Started

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally: `git clone https://github.com/your-username/sequelize-ai.git`
3. **Install dependencies**: `npm install`
4. **Create a branch**: `git checkout -b feature/your-feature-name`

## Development Workflow

- The core logic lives in `src/`.
- `examples/run.js` contains a mock database and complex query examples you can use to test logic locally.
- Run tests using Vitest: `npm test`
- Make sure to format your code: `npm run lint:fix && npm run format`

## Pull Request Process

1. Ensure all tests pass (`npm test`).
2. Update the README or Docs if you are adding new features.
3. Open a Pull Request with a clear title and description.
4. An maintainer will review your code.
